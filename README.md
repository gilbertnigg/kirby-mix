# Kirby-Mix
### Boilerplate for Kirby CMS &amp; Laravel Mix &amp; Tailwind CSS  

Install frontend  
- [Laravel Mix](https://laravel-mix.com  )
- [Tailwind CSS](https://tailwindcss.com )
- [BrowserSync](https://browsersync.io/)
```
npm i
```

Install backend  
- [Kirby CMS](https://getkirby.com  )
```
cd public
composer install
```
